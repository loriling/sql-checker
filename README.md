# sql-checker 
一个基于nodejs的数据库检测定时服务

## Base On

- nodejs
- yarn
- 如果需要连接oracle，需要安装相关客户端程序 [node-oracledb](https://oracle.github.io/node-oracledb/INSTALL.html#quickstart)

##Configuration

- log.json 用来配置日志相关信息
- config.js 用来配置定时与数据库相关信息

## Start

 - Clone 或者下载这个repo.gitignore
 - 进入目录，安装依赖

``` bash
yarn
```

###Run

```bash
node main.js
```
